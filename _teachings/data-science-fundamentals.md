## Course Overview

By the end of this course, students will be able to:

- Define the development process of an agent-based system
- Adopt the most appropriate architecture for a software agent
- Design goal-oriented autonomous software agents
- Apply automated planning techniques for agent deliberation
- Implement autonomous software agents, including LLM-based agents
